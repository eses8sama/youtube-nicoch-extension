# 📺 YouTube Nico-style Chat Extension
This browser extension displays YouTube Live chat comments as scrolling text across the screen, similar to Nico Nico Douga. It is highly customizable to suit your preferences.

- **Speed Control:** Adjust the scrolling speed of the comments.
- **Text Customization:** Change size and opacity so it doesn't distract from the video.
- **Display Area:** Specify the number of lines displayed simultaneously.
- **Test Message:** Instantly preview your design settings.
- **Reset Feature:** Easily revert to default settings at any time.

![Settings UI](settings.png)

## 🚀 How to Install

Since this is a development version (not on the official store), please follow these steps to install it manually:

1. **Download the files**
   - Click the green **"Code"** button at the top right and select **"Download ZIP"**.
   - Right-click the downloaded ZIP file and select "Extract All".

2. **Open the Extensions page**
   - Open Chrome or Edge and type `chrome://extensions/` in the address bar.

3. **Enable Developer Mode**
   - Toggle the **"Developer mode"** switch to **ON** (usually in the top right).

4. **Load the extension**
   - Click the **"Load unpacked"** button.
   - Select the folder you just extracted (the one containing `manifest.json`).

5. **Done!**
   - Open a YouTube Live stream, and comments will start scrolling across the screen.
